`/` 
- in linux like `C:/` in Windows
- contains file system
#### Note
> Two things in linux we call as **root**
> - **/** file system
> - **super user:** that has all permission 

## The folders in /

![[Picture1.png]]
- `etc` folder: contains all settings like, clock, networks, etc.
- `sbin`: contains commands that is owned by super user. Only super user can run them
- `home`: contains all users that have been created
- `root`: contains the super user (as `home`)
