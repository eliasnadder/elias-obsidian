### I- Create a script
1. Create a script
```bash
vi script
```

2. First line in script should be
```bash
#!/bin/bash
```
	To determine the shell

3. Make file executable
```bash
chmod +x <file>
```

> [!NOTE] Note
> Or you execute it by using
> ```bash
> bash ./<file>
> ```
